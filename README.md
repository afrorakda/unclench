# UNCLENCH | Fridge Combo #48

A hyper-minimalist web tool designed to combat subconscious jaw clenching (Bruxism/TCH) through visual biofeedback and timed relaxation.

## 1 Action / 1 Tool
- **Action**: Hold the central button for 3 seconds.
- **Experience**: The background breathes from white to a soothing green, guiding you to physically release your jaw and teeth.

## Core Philosophy
- **Awareness over Data**: No tracking, no logs. The tool acts as a "digital sticky note" to remind you to relax in the moment.
- **Privacy First**: No login, no ads, browser-only.
- **Minimalist Design**: Adheres to the Fridge Combo Design System (#fff background, #333 elements, 15px border-radius).

## Features
- **Visual Breathing**: Background color transition synchronized with a 3-second hold.
- **Haptic Guidance**: Prevents shallow engagement by requiring a deliberate 3-second press.
- **Optimized for Mobile**: iOS/Android friendly UI with prevention of accidental zoom and system text-fill overrides.

## How to use
1. Open the tool when you feel stress or tension.
2. Hold the button and follow the instruction: "Drop your jaw."
3. Feel the tension melt away as the screen turns green.
